# 如何使用
```
git clone https://github.com/ZhenJie-Zhang/line_chatbot_quick_template.git
cd line_chatbot_quick_template
sudo docker-compose up -d
sudo docker exec -it line-chat-bot-jupyter bash
jupyter notebook --allow-root
```
